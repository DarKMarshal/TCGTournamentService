# TCG Tournament Service

An **online tool** for card shop owners to **organize, manage, and share** their store tournament results.

---

## What This Is

A web based system for tracking and sharing the tournament results that are uploaded from the output from tournament software that is used for the events. The system parses the results, and displays them in a player friendly view for the public.

The system is designed for **avalibility of results** for communities that do not have reporting.

---

## Scope (In progress)

- Full parsing and online display of results from the TOM software
- Player accounts for personal tracking
- Store accounts for store specific actions
